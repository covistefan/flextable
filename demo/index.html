Options

Option Name Option Values More Info
bt string 'parent', 'window' BreakType: 'parent' » parent element width, 'window' » based on window width
bw int 768 BreakWidth: int » only with bt: 'window', if window width is lower then bw
cm string 'auto', 'load' CalcMode 'auto' » on load and on resize, 'load' » only on load
hparent string '' selector of parent holding header cells to get tablehead if not defined by thead » used if thead not found AND hcell is empty
hcell string '' selector of cells to get tablehead if not defined by thead » used if thead not found
header string 'before', 'after', 'none' showing tablehead information 'before' » add li with header data before data cell, 'after' » add li with header data after data cell, 'none' » ignore header data
combine bool false false » displays data as simple list, true » creates a '<li><ul><li>header</li><li>data</li></ul></li>' set if header is included
debug bool false Debug mode, shows some information in js console
